## How to create the landing page
These steps are found in the GitHub docs webpage under **"GitHub Pages"**.

1. Create a new repository with a README file.

2. Under your repositry name, click on the settings icon.

3. Under **"Code and automation"** section, click on **"Pages"**.

4. Then under **"Build and deployment"**, click the dropdown menu that says **"Source"** and select **"Deploy from a Branch"**.

5. Under** "Build and deployment"**, under the **"Branch"** dropdown menu, select your publishing source.

6. Here you can decide whether you want to use whatever css or html files you created for your landing page, or alternatively you can publish from your **README**.

7. From there you can access the website a few ways:
  1. From the **About** section in a repo, under **Website**.
  2. Under the "Pages" section in the repo settings the website link will be at the top of the page.
  3. You can also search for the website by searching the organization name, followed by .github.io, then repo name as shown below
    `mirrulations.github.io/mirrulations-landing-page`
